# 🛒 Retail Sales Data Analysis & Prediction

## 📌 Project Overview
This project performs real-world retail sales data analysis using Python and Machine Learning. The goal is to uncover business insights, analyze sales trends, and predict future sales.

---

## 🎯 Objectives
- Perform Exploratory Data Analysis (EDA)
- Analyze sales and profit trends
- Identify top-performing categories
- Visualize business insights
- Predict sales using Machine Learning

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

---

## 📂 Dataset
Dataset used:
- Sample Superstore Dataset

Dataset includes:
- Sales
- Profit
- Region
- Category
- Quantity
- Discount

---

## 📈 Analysis Performed

### ✅ Data Cleaning
- Removed duplicate values
- Checked missing values

### ✅ Data Visualization
- Sales by Category
- Profit by Category
- Region-wise Sales
- Monthly Sales Trend
- Correlation Heatmap

### ✅ Machine Learning
- Linear Regression Model
- Sales Prediction
- Model Evaluation using R² Score

---

## 🔍 Key Insights
- Technology products generated the highest sales.
- Discounts impact overall profitability.
- Some regions outperform others in sales.
- Profit and Sales are positively correlated.
